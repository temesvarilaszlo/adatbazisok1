1.    Melyek azok a hajók, amelyeket 1921 előtt avattak fel?

	π hajonev σ felavatva < 1921 Hajok

	select hajonev from Hajok where felavatva < 1921;

	pi osztaly, orszag σ kaliber >= 16 Hajoosztalyok

	select osztály, ország from Hajóosztályok where kaliber >= 16;

13.    Adjuk meg azokat az osztályokat, amelyekbe csak egyetlenegy hajó tartozik.

	σ hajok=1 γ osztaly; count(*)->hajok Hajok

	select osztály, count(*) from Hajók
	group by osztály
	having count(*) = 1;

14.    Évenkénti bontásban hány hajót avattak?

	γ felavatva; count(*)->count Hajok

	select felavatva, count(*) from Hajók
	group by felavatva;

15.    Mely hajóosztályból mikor avatták az utolsó hajót?

	γ osztaly; max(felavatva)->hajok Hajok

	select osztály, max(felavatva) from Hajók
	group by osztály;
